# Thapar Mandi

A full-stack campus marketplace built for Thapar University students to buy, sell, and discover products within the campus community.

## Live Demo

https://thaparmandi.store

## Features

* User authentication
* Email OTP verification using Resend
* Create and manage product listings
* Browse products posted by other students
* Search and discover listings
* Responsive user interface
* Secure backend and database integration

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database & Services

* PostgreSQL
* Supabase
* Resend (Email OTP Verification)

## Why I Built This

As a first-year student, I noticed there wasn't a dedicated platform where students could easily buy and sell items within the campus community. Thapar Mandi was built to solve that problem while helping me learn full-stack development, database management, authentication, deployment, and real-world software engineering practices.

## Key Learnings

* Building and deploying a full-stack application
* Authentication and email verification workflows
* Database design and integration
* API development with Express
* Managing production deployments
* Working with modern web technologies
